Hangman WebService API

The WSDL for this service can be found at:

http://10.100.106.19:8080/hangmanWebService/hangman?WSDL

The webservice offers the following functionality:

Method Name Method Parameters Return Values
checkLogin String username, String password 0 if credentials incorrect, -1|-2 if problem talking to database, positive integer referencing users.autoID if correct
registerUser String name, String surname, String username, String password, String email 0 if cannot add credentials, -1 if problem talking to database, positive integer referencing users.autoID if correct
newGame int playerID Returns the id (games.autoID) of the game just created, otherwise it returns -1
getWord int gameId the word this game is using
addGuess String the letter guessed, int playerId, int gameId a comma separated list of indices where the letter matches or, "" (an empty string) if it didn't match, otherwise "EXCEPTION_ERROR"
guessesUsed int gameId the number of guesses made
getRandomWord NO PARAMETERS REQUIRED returns a random word from the database, occasionally returns an empty string
